Rust 运算符

关键字: Rust 运算符 特性 重载

在 Rust 里, 有很多不一样的操作符, 类似 + - * / 运算符, 每个运算符在不同的应用条件下表达的意思也可能不一样, 而 Rust 中的运算符很特殊, Rust 中的运算符支持重载, 运算符重载这个特性在大部分语言中都没有得到支持, 而 Rust 支持了这一特殊的特性.
